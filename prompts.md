# SuryaSetu — Image Generation Prompts · DARK THEME (Firefly / nano-banana)

Heavily-engineered prompts for the SuryaSetu pitch-deck diagrams and hero art (ISRO BAH 2026,
PS-15). **This file is the DARK theme** (deep-space navy ground, glowing accents) — it matches
the live Vercel app (dark starfield). For the **LIGHT theme** that matches the pitch-deck
slides, use `prompts-new.md`. The 4 hero/space images are identical in both files (space is
dark by nature).

> **Recommended mapping:** dark `prompts.md` → Vercel app · light `prompts-new.md` → pitch deck
> slides · hero art → shared.

---

## How to use this file
- **Target models:** Adobe Firefly and nano-banana (Gemini image). Paste a full prompt block as-is.
- **One prompt = one image.** Every block is self-contained (no placeholders to fill).
- **Text inside images:** image models can still misspell labels. After generating, **read every
  label**; if any word is wrong, regenerate or fix it in PowerPoint. For dense images (dashboard,
  architecture) prefer generating the visual and overlaying final text in PowerPoint.
- **Output:** PNG, saved into `/images`. Naming: deck diagrams = `slideNumber_imageNumber.png`
  (e.g. `5_1.png`); hero art = `app_<name>.png`.

## Design system (the SuryaSetu look — DARK)
Bake this identity into every diagram so the set feels like one product, not nine clip-arts.

- **Ground:** deep-space navy `#070a12`. **Panels/cards:** `#111a2e` and `#16203a`.
  **Hairline borders:** `#233150` (1px, crisp).
- **Role-based accent colors (use consistently):**
  - Hard X-ray / primary action = **orange `#ff6b1a`**
  - Headings / emphasis = **amber `#ffa726`**
  - Soft X-ray / science = **blue `#4aa8ff`**
  - Quiet / good / catalogue = **green `#34d399`**
  - Precursor / derivative / ML = **purple `#a78bfa`**
  - Alert = **red `#f0506e`**, Watch = **yellow `#fbbf24`**
  - Body text = `#e8edf7`, muted/secondary = `#93a0b8`
- **Typography:** clean geometric humanist sans (Segoe UI / Inter feel). Bold headings,
  regular body, and **monospace (Consolas/JetBrains Mono feel) for ALL numbers, timestamps,
  keV ranges, and table data.**
- **Line & shape system:** rounded-rectangle nodes (6–10px radius), 1.5px borders in the node's
  role color, thin 2px connector lines, neat triangular arrowheads, consistent icon weight.
- **The "spark" (premium finish):** subtle neon glow / bloom around accent strokes and key
  nodes; gentle inner depth on panels; a barely-there dot grid in the background; layered depth
  (focal node slightly brighter, rim-lit). Think Stripe / Linear / Apple-keynote / ISRO
  mission-control graphics: flagship, intentional, generous spacing, zero clutter.
- **Completeness rule:** every listed node must appear and be labeled — no empty boxes, no
  "lorem", no dropped stages. Align everything to an invisible grid; balance the negative space.

## Removing the Gemini / nano-banana watermark
nano-banana (Gemini) stamps a **visible sparkle logo** (bottom-right) *after* generation, plus an
invisible SynthID mark. Putting "watermark" in the negative prompt does **not** remove the visible
one. Options:
1. **Generate with margin, then crop** — add "leave a clean empty margin around all edges" to the
   prompt, then crop the corner strip (re-check the aspect ratio after cropping).
2. **Use Adobe Firefly for finals** — paid plans have no visible watermark.
3. **Cover it** — drop a solid `#070a12` rectangle over the corner; the flat navy ground hides it
   invisibly.
SynthID is invisible and has zero visual impact — leave it.

## What is NOT here (kept code-generated)
The 5 precise data charts stay generated by `build_diagrams.py` to preserve scientific accuracy
and the "renders real Aditya-L1 data" claim — do **not** AI-generate these: `real_lightcurve.png`,
`spectrum.png`, `neupert.png`, `feature_importance.png`, `tss.png`.
